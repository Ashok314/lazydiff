# Privacy Policy

LazyDiff does not collect, transmit, sell, or share user data.

## Data Handling

LazyDiff runs locally in your browser on GitHub pull request pages.

- No analytics.
- No tracking.
- No remote servers.
- No network requests from the extension.
- No source code, file paths, filter text, or pull request content is sent anywhere.

## Local Storage

LazyDiff stores your filter text in `chrome.storage.local` so the filter can persist between page reloads.

This data stays in your browser's local extension storage. It is not uploaded by LazyDiff.

## Permissions

LazyDiff requests:

- `storage`: used only to save filter text locally.

LazyDiff runs only on GitHub pull request pages matched by the extension manifest.

## Contact

For privacy questions, open an issue in the LazyDiff repository.
