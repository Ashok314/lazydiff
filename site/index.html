Screenshots

Custom syntax

One box for include and exclude.

apps

Show matching paths.

!apps

Hide matching paths.

!/apps

Hide root paths only.

apps !*.spec.ts

Show apps, except specs.

Private by default

No network. No analytics. No code leaves your browser.

LazyDiff runs locally on GitHub PR pages. Filter text is stored only in Chrome local storage. See the privacy policy.

Release plan

Chrome Web Store listing coming after final QA.
